## Requiriments

Requires Neovim (>= v0.8.3)

```bash
# Npm packages
npm i -g tree-sitter-cli typescript tree-sitter fd fd-find

# System packages
ripgrep xsel unzip curl gcc zip unzip gzip g++ lazygit xclip wl-clipboard rustup clang
```
