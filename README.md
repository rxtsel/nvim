# My neovim setup

This is simple and minimal, but powerful neovim setup.

<img width="2212" height="1361" alt="thumb" src="https://github.com/user-attachments/assets/ec7a8809-853f-4410-b67f-93de4d07adfb" />

**Requiriments**:

- Neovim >= `v0.12`

```bash
# Nodejs packages
npm install -g tree-sitter-cli tree-sitter

# System packages
ripgrep gcc clang jq fd wl-clipboard unzip tar curl lsof imagemagick

# LSP servers
astro-language-server lua-language-server nixd svelte-language-server vtsls copilot-language-server tailwindcss-language-server vscode-css-languageserver oxlint marksman

# Formatters
alejandra stylua rustfmt biome prettierd oxfmt
```
